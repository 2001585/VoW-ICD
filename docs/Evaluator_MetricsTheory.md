# Evaluator Component Specification

본 문서는 Village of Words 실험의 정량 평가 지표 설계를 명시한다. 각 지표는 기존 협력 연구에서 출처를 갖고 있으며, 본 연구에서는 이를 '협력 형성–유지–붕괴' 메커니즘에 맞게 통합 적용하였다.

---

## 평가 지표 목록 및 출처 주석

### 1. 협력 행동 비율 (Cooperation Ratio)
- **정의**: 집단 목표에 평균 이상 자원을 기여한 행동 비율
- **형성 단계 측정**에 해당
- **출처**: Public Goods Game 기반 실험 (Kulkarni & Brunswicker, 2024)
  - 해당 논문에서는 각 LLM의 기여 여부로 협력 여부를 평가
- **본 연구 적용 방식**: 에이전트의 발화 + 행동 결과(JSON)에 기반한 자동 판별

### 2. 총 기여량 (Total Contribution)
- **정의**: 각 턴마다 전체 에이전트가 공동 목표에 낸 자원의 총합
- **형성 & 유지 단계 측정**에 사용
- **출처**: 동일 PGG 실험 (위와 동일)
- **특이점**: 기여량의 변화를 통해 협력 붕괴 전후 비교 가능

### 3. 협력 지속 시간 (Avg. Duration of Cooperation Segment)
- **정의**: 협력 상태가 연속 유지된 구간의 평균 길이 (턴 단위)
- **유지 단계 핵심 지표**
- **출처**: 없음 – 본 연구에서 최초 정의
  - 기존 실험에서는 '연속 협력 유지'를 별도로 추적하지 않음

### 4. 회복 시간 (Recovery Time)
- **정의**: 협력이 붕괴된 후 다시 회복되기까지 걸린 턴 수
- **붕괴 이후 복구 단계 측정용**
- **출처**: 없음 – 본 연구 독립 정의
  - Generative Agents 등에서도 협력 회복을 수치화하지 않음

### 5. Gini 계수 (Gini Coefficient)
- **정의**: 에이전트 간 자원 기여의 불균형 측정
- **형성/유지 모두에 걸친 형평성 판단용**
- **출처**: MultiAgentBench, 경제학 기반 분배 실험들
- **본 연구 적용 방식**: 각 턴 기여량을 벡터로 변환 후 계산

### 6. 대화량 편중도 (Dialog Centrality Score)
- **정의**: 특정 에이전트가 전체 대화량에서 차지하는 비율
- **형성/유지에 간접 영향 미치는 변수**
- **출처**: 일부 다중 대화형 AI 실험, 네트워크 중심성 분석 논문 (2024)
- **본 연구 적용 방식**: JSON 메시지 로그 기준 발화 수 기준 비율 계산

---

## 향후 확장 가능 지표 (추가 제안)

### 7. 배신 이벤트 비율 (Betrayal Rate)
- **정의**: 발화상 협력을 약속하고 행동상 불참한 비율
- **출처**: Inspired by LLM negotiation dishonesty detection (Abdelnabi et al., 2024)
- **적용 예정**: `message`와 `decision` 간 불일치 자동 탐지 로직과 연계

### 8. 에이전트 신뢰도 스코어 (Trust Score)
- **정의**: 과거 발화/행동의 일치성 추적 기반 누적 스코어
- **출처**: 협상 시뮬레이션 모델 중 신뢰 점수 평가 실험
- **적용 조건**: 메모리/기억 기능 구현 이후 적용 가능

---

본 문서의 지표 정의는 Village of Words 실험 설계의 서론 및 관련 연구 구조와 일관되며, 기존 논문 기반의 정량화 항목을 '협력 메커니즘' 흐름에 통합한 형태로 재설계되었음. 이로써 LLM 기반 에이전트 실험에서 최초로 형성–유지–붕괴 메커니즘을 자동 추적 가능한 방식으로 수치화한 기반을 마련한다.
