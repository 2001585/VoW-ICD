2.1 Generative Agents
 Park et al. (2023)은 대규모 언어 모델을 활용해 자율적인 가상 에이전트들이 하나의 마을에서 생활하는 시스템을 제안
하였다. 해당 연구에서는 LLM 기반 캐릭터들이 기억과 계획, 일과를 가지며, 자연스러운 사회적 상호작용을 수행할 수
있음을 시연하였다. 장기 기억 구조, 행동 생성 메커니즘, 일상적 이벤트 처리 루프 등을 포함하며 LLM을 자율적 사회
행위자로 구성한 대표 사례다. 하지만 상호작용의 대부분은 내러티브 서술로 정성적으로 기술되었고, 협력이나 사회적
구조의 정량적 분석은 이루어지지 않았다. 또한 시각적 환경에 의존한 인터페이스로 인해 CLI 기반의 구조적 평가 실험
에는 적합하지 않다.
 2.2 MetaGPT
 Hong et al. (2023)은 LLM 에이전트들이 조직 내부에서 각자 역할(CEO, PM, Engineer 등)을 수행하며 하나의 소프트
웨어를 공동 개발하는 환경을 제안하였다. 각 역할에 적합한 명시적 프롬프트를 부여하여 협력형 분업 구조를 구성하였
으며, 실제로 코드 생성, 검토, 통합까지 수행하는 일련의 협력 프로세스를 재현하였다. 이 연구는 역할 기반 협력의 정
량성과 자동화 구조에 있어 장점이 있으나, 내부 갈등, 배신, 협력 붕괴 등 비선형적 사회 메커니즘은 설계에 포함되어
있지 않다. 또한 각자의 행동은 중앙 조정자(PM)의 존재 하에 이루어져 자율적 협상이나 관계 형성은 부재하다.
 2.3 MultiAgentBench
 MultiAgentBench (OpenAGI, 2025)는 다중 LLM 에이전트를 평가하기 위한 벤치마크 프레임워크로, 다양한 시나리
오에서 에이전트 간 상호작용을 정량화할 수 있도록 구성되어 있다. 해당 연구에서는 대화 분량, 합의 도출 성공률, 임무
완수율, 상호 협상 결과 등의 수치를 통해 LLM 협업의 품질을 수치화하였다. 본 연구가 제안하는 평가 지표(Gini 계수,
협력 지속 시간, 회복 시간 등)는 이 프레임워크의 일부 구성에서 착안하였다. 다만 MultiAgentBench는 대화 중심 과
업을 중심으로 설계되어 있어 장기적인 협력 구조의 형성 및 붕괴 메커니즘은 평가하지 않는다.
 2.4 Public Goods Game 기반 LLM 실험
Kulkarni & Brunswicker (2024)는 전통적인 공공재 게임을 LLM 기반 시뮬레이션 환경에 적용하였다. 각 LLM 에이전
트는 기여 또는 무임승차를 선택하며, 에이전트의 기여 수준, 집단의 총 자원 형성, 게임 종료 시점에서의 분배 결과 등
을 통해 협력 행동을 수치화하였다. 본 연구의 실험 구조는 이와 유사한 ‘자원 분배와 협력’ 메커니즘을 포함하되, 자연
어 상호작용과 정제된 JSON 출력을 통해 보다 구조적인 정량 분석이 가능하도록 구성되었다. Kulkarni의 연구는 게임
이론적 프레임워크를 AI 시뮬레이션에 적용한 선례로서 실험 타당성의 근거를 제공한다.

득점(Own Score), 집단 득점(Collective Score), 실패율(Wrong Deal Rate) 등이 있으며, 협상의 효율성과 상호 수용
