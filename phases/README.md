# Phase Folder Guide

각 Phase 진행 시 이 디렉터리의 하위 폴더에 작업 로그, 임시 산출물, 체크리스트 스냅샷을 저장합니다. 정식 명세는 `docs/PHASE.md`를 따르며, 아래 요약은 빠른 참조용입니다.

- `phase-01/`: 환경 구축 및 구조 점검 자료 (예: 설치 로그, 구조 캡처, 스펙 문서 초안 메모)
- `phase-02/`: 에이전트 코어 구현 초안, 테스트 코드 초안
- `phase-03/`: 턴 매니저, 실행 파이프라인 관련 산출물
- `phase-04/`: 지표 계산 및 리포트 자동화 결과물
- `phase-05/`: 확장 실험, 커버리지/품질 개선 메모

Phase 완료 후에는 관련 문서/스크립트를 정리해 상위 디렉터리로 승격하고, 이 폴더에는 임시/중간 결과만 남깁니다.
